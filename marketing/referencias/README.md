# Referências visuais — Ligeirinho Marketing

Coloque aqui imagens de referência (prints, artes da TV, listas do WhatsApp, telas atuais, etc.) para orientar o desenvolvimento do app Marketing.

## Estrutura sugerida

| Pasta | Use para |
|-------|----------|
| `tv/` | Layout da TV na loja, slides, loop de promoções |
| `promocoes/` | Cards de promoção, listas do dia, preços em destaque |
| `whatsapp/` | Lista + imagem enviada no grupo, formato atual |
| `geral/` | Outros materiais (logo, cores, exemplos diversos) |

## Formatos aceitos

PNG, JPG, JPEG, WebP e GIF.

Prefira nomes descritivos, por exemplo:

- `tv-loja-tela-cheia.png`
- `lista-promocoes-dia-25.jpg`
- `whatsapp-cantinho-do-vale.png`

## Observação

Esta pasta é só para **referência de design** no repositório. As imagens finais de produtos e promoções no sistema virão do catálogo e do cadastro no app (Supabase), não desta pasta.
